# 竞品价格监控 · 方案文档（v1）

> 对应 Task #9（模块设计 + SKU 配置）。代码已落地在 `utils/price_monitor/`。

## 1. 目标与范围

监控海南主要免税店**公开网页商城**的**香水主力 SKU**竞品标价，用于：
- 跨零售商横向比价（中免 vs 海旅 vs 海控 vs 中服）
- 跟踪单 SKU 历史价格走势
- 降价预警（较上一次快照跌 ≥5% 标红）

**v1 覆盖（已与 MAX 确认）**：中免 CDF、海旅免税、海控 GDF会员购、中服 CNSC，共 4 家 × 30 个香水旗舰 SKU。

## 2. 合规性（红线，不可突破）

| 规则 | 说明 |
|------|------|
| 仅抓公开标价 | 只读取无需登录即可见的商品价格页 |
| 拒绝登录墙 | 命中「登录/注册」遮罩 → 标记 `blocked_login` 并跳过，**绝不抓取会员/登录专属价** |
| 拒绝小程序 | 微信小程序（加密 API）一律不碰 —— 王府井即因此被排除 |
| 限速 | 每条之间间隔 ≥10 秒，随机 UA，单页超时 30s |
| 失败重试 | 单条最多重试 3 次，仍失败标 `error`，不拖垮整轮 |
| 来源标注 | 每条记录强制带 `source` 标签（如「中免CDF官网公开标价」） |

## 3. 技术架构（包结构）

```
utils/price_monitor/
├── __init__.py
├── config.py          # 零售商注册表 + 香水SKU清单 + 合规护栏（唯一静态配置）
├── models.py          # PriceRecord 数据模型 + CSV 表头
├── fetcher.py         # 采集编排：限速/重试/双写/历史读取（主入口）
└── adapters/
    ├── __init__.py    # ADAPTER_REGISTRY（id→适配器类）
    ├── base.py        # BaseAdapter 模板方法 + 共用逻辑
    └── generic.py     # GenericH5Adapter（中免/海旅可直接用）
```

**数据流**：`fetcher.run_monitor()` → 逐个零售商 × SKU → 适配器 `run()`（导航→判登录墙→读价）→ `PriceRecord` → 双写本地 `data/price_monitor/{日期}.csv` + `latest.csv` 与 OSS（`price_monitor/...`）。

## 4. 抓取引擎选型

**Playwright 无头浏览器**（非逆向后台签名接口）。理由：
- 中免/海旅是 uniapp H5 单页应用，价格由 JS 渲染，无头浏览器读渲染后 DOM 最稳。
- 实测盲猜中免后台接口返回 404（签名/路径不明），逆向脆弱易失效。
- **目标部署 = 阿里云轻量应用服务器**：`playwright install chromium` 原生可装，cron 定时也原生支持，比在 Streamlit Cloud 免费档折腾浏览器可靠。

| 档 | 零售商 | 结论 | v1 处理 |
|----|--------|------|---------|
| A | 中免 CDF `m.cdfhnmall.com` | ✅ 公开标价、无需登录（实测 200） | GenericH5Adapter 直接抓 |
| A | 海旅免税 `m.hltmsp.com` | ✅ 公开标价、无需登录（实测 200） | GenericH5Adapter 直接抓 |
| B | 海控 GDF会员购 `shop.gdfplaza.com` | ⚠️ 会员商城，curl 超时，可能卡登录墙 | 先验证；命中登录墙 → `blocked_login` 跳过（#10 精调） |
| B | 中服 CNSC `www.zhongfusanya.com` | ⚠️ 线上入口不明，疑似小程序为主 | 先验证；入口不明 → `verify_pending`（#10 精调） |
| C | 王府井 | ❌ 仅跨境小程序，无公开网页商城 | 已排除（合规红线） |

## 6. v1 看板设计（页面 `pages/sell_out/13_竞品价格监控.py`，Task #11）

沿用「商情监控看板」视觉与防御性导入风格：
1. **最新快照表**：行=SKU，列=各零售商价格；来源标签 + 抓取时间；`blocked_login`/`verify_pending` 灰显，`not_found` 标「—」。
2. **降价预警区**：对比上一次快照，跌幅 ≥5% 的 SKU 标红 + 「↓ 降幅%」。
3. **历史趋势**：点选 SKU → 折线图（各零售商价格随时间）。
4. **数据下载**：CSV（含 is_estimate=False 明确区分实测）。
5. **立即抓取一次**按钮：调用 `run_monitor()`（需服务器已装 chromium）。
