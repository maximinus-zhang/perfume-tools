# 竞品价格监控 · 方案文档（v1）

> 对应 Task #9（模块设计 + SKU 配置）。代码已落地在 `utils/price_monitor/`。

## 1. 目标与范围

监控海南主要免税店**公开网页商城**的**香水主力 SKU**竞品标价，用于：
- 跨零售商横向比价（中免 vs 海旅 vs 海控 vs 中服）
- 跟踪单 SKU 历史价格走势
- 降价预警（较上一次快照跌 ≥5% 标红）

**v1 覆盖（已与 MAX 确认）**：中免 CDF、海旅免税、海控 GDF会员购、中服 CNSC，共 4 家 × 30 个香水旗舰 SKU。

## 2. 合规性（红线，不可突破）

| 规则 | 说明 |
|------|------|
| 仅抓公开标价 | 只读取无需登录即可见的商品价格页 |
| 拒绝登录墙 | 命中「登录/注册」遮罩 → 标记 `blocked_login` 并跳过，**绝不抓取会员/登录专属价** |
| 拒绝小程序 | 微信小程序（加密 API）一律不碰 —— 王府井即因此被排除 |
| 限速 | 每条之间间隔 ≥10 秒，随机 UA，单页超时 30s |
| 失败重试 | 单条最多重试 3 次，仍失败标 `error`，不拖垮整轮 |
| 来源标注 | 每条记录强制带 `source` 标签（如「中免CDF官网公开标价」） |

## 3. 技术架构（包结构，已落地）

```
utils/price_monitor/
├── __init__.py
├── config.py          # 零售商注册表 + 香水SKU清单 + 合规护栏（唯一静态配置）
├── models.py          # PriceRecord 数据模型 + CSV 表头
├── fetcher.py         # 采集编排：引擎分发/限速/重试/双写/历史读取（主入口）
└── adapters/
    ├── __init__.py    # ADAPTER_REGISTRY（id→适配器类）
    ├── base.py        # BaseAdapter（浏览器模板方法）+ ApiBaseAdapter（纯HTTP）+ http_get_json()
    ├── generic.py     # GenericH5Adapter（仍需浏览器的零售商，如海旅）
    └── cdf_api.py     # CdfApiAdapter（中免，纯 HTTP，无需浏览器）
```

适配器分两类引擎（由 `ENGINE` 类属性区分）：
- `ENGINE="api"`：**纯标准库 `urllib` HTTP 适配器**，直接调商城公开搜索接口，无需浏览器、无 chromium 依赖。
- `ENGINE="playwright"`：仍需无头浏览器渲染的适配器（如海旅）。

**数据流**：`fetcher.run_monitor()` 先判断是否存在 playwright 适配器 → 有才启动浏览器；逐零售商 × SKU 调适配器 `run()` → `PriceRecord` → 双写本地 `data/price_monitor/{日期}.csv` + `latest.csv` 与 OSS（`price_monitor/...`，无凭证静默降级）。

## 4. 抓取引擎选型（已转向纯 HTTP）

**中免：放弃 Playwright，改用纯 HTTP 搜索接口**（实测更稳、零浏览器依赖）。
- 真实接口：`GET https://service.cdfhnmall.com/mini/findGoodsList`，需 headers `appversion:10.12.60`、`channel:big_frontend_h5`、`token:""`（空=公开，无需登录）。
- 价格口径：取 `goodsPrice.showPrice`（实际展示免税价）；`salesPrice`=有税参考、`estimatePrice`=预估。缺 showPrice 时诚实标注「取有税参考」。
- 价值：可在 **Streamlit Cloud 免费额度 / 任意无 chromium 环境**跑，部署门槛大幅降低。
- 海旅同思路（uniapp H5，应可拆纯 HTTP，待 #10 后续）：需像中免一样抓真实搜索请求，替换 GenericH5Adapter。

## 6. v1 看板设计（已落地 `pages/sell_out/13_竞品价格监控.py`，Task #11）

沿用「商情监控看板」视觉与防御性导入风格，且**多零售商通用**（新零售商接入自动生效）：
1. **当前快照表**：行=SKU，列=零售商/品牌/产品/规格/价格/状态/抓取时间/来源/备注；降价 ≥5% 红色加粗，无价灰显。
2. **降价预警区**：对比上一次快照，跌幅 ≥5% 的 SKU 汇总提示（含 ¥前→¥后 + 降幅%）。
3. **历史趋势**：点选 SKU → 折线图（跨日价格，load_history 聚合）。
4. **数据下载**：CSV（含 is_estimate=False 明确区分实测）。
5. **立即抓取一次**按钮：后台线程跑 `run_monitor`（仅 API 零售商即中免，免浏览器），完成后自动刷新；多零售商就绪后可扩展。
6. **合规说明**：页脚明示「全为实测公开标价、非估算」「仅抓公开页、不碰登录墙/小程序」。
